Research repository for the comparative evaluation of Earth Embeddings and Geospatial Foundation Models for spatial prediction tasks.

## Project Overview

This research project investigates the use of Earth Embeddings and Foundation Models for geospatial prediction across multiple domains, including:

- Built environment
- Infrastructure
- Public goods
- Poverty estimation
- Crime prediction
- Economic activity
- Accessibility analysis

The project focuses on benchmarking and comparing models such as:

- AlphaEarth Foundations
- Clay
- Prithvi
- SatCLIP
- GeoCLIP
- Vision Transformers (ViT)

## Research Objectives

- Understand Earth Embedding architectures
- Study self-supervised learning approaches
- Compare embedding quality across models
- Fine-tune open-source geospatial foundation models
- Evaluate transferability and robustness
- Develop multimodal geospatial prediction pipelines
